# 🧠 Trained Break the Bias

**濱口秀司（Hideshi Hamaguchi）の「Break the Bias（バイアス壊し）」メソッドを実践するためのWebアプリ。**

## 使い方

1. **テーマを決める** — 新しいアイデアが欲しい領域を設定
2. **2つの軸を設定する** — 業界の「暗黙の前提」を軸にして二軸マトリックスを作成
3. **アイデアをマッピング** — 各マス目をタップしてアイデアを追加。密集した場所 = バイアス
4. **「バイアスを壊す」モード** — 空白地帯に強制発想。極端で非現実的なアイデアを！
5. **分析レポート** — 思考の偏りを可視化してインサイトを得る

## 開発

```bash
# ローカルで開くだけ
open index.html
```

## ベースとなったメソッド

- 濱口秀司氏「Break the Bias」 — 二軸マトリックスでバイアスを可視化し、空白地帯に強制発想する
- 参考文献：『SHIFT：イノベーションの作法』、Biz/Zine連載、Hills Life対談
